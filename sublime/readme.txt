Setup:

1. Install Package Control: https://packagecontrol.io/installation.
2. Open `Preferences > Browse Packages` and copy `Preferences.sublime-settings`
   into the `User/` directory there.
3. Download the fonts from the Preferences file however you download fonts (and
   restart Sublime after installing).

Packages:

- A File Icon
- DocBlockr
- GitHubinator
- Missing Palette Commands
- Package Control
- Pretty JSON
- Python Improved
- SFTP
- SideBarEnhancements

With language-/project-specific packages added as they're needed:

- LSP: https://lsp.sublimetext.io
- GoSublime
- Rust Enhanced
- etc.
